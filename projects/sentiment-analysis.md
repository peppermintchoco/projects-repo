# Game Sentiment Analysis Using Steam Reviews

## Overview
This project explores the Steam Store Games (Cleaned Dataset) on Kaggle, with the aim of exploring the game features that could result in greater positive overall player sentiment.

## Objectives
- Data cleaning and feature engineering
- Eploratory data visualization and analysis
- Predictive modelling
- Model evaluation using mean absolute error (MAE) values and SHAP values 

## Methods & Tools
- Python, Pandas, NumPy 
- Scikit-learn (XGBoost) 
- Jupyter Notebook for analysis and documentation

## Key Findings
- The model achieved a MAE value of 0.16.
- Game price had the greatest influence on sentiment prediction with a mean absolute SHAP value of 0.0255.
